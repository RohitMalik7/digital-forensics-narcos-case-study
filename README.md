# Digital Forensics Investigation - Narcos Case Study

> **Author:** Rohit Malik | **Domain:** Digital Forensics | **Type:** Academic Project | **Purpose:** Portfolio & Educational

---

## Overview

This project presents a comprehensive Digital Forensics and Incident Response (DFIR) investigation conducted on a simulated cybercrime case involving cross-border drug trafficking.

The analysis focuses on examining disk images and memory dumps from multiple Windows systems to reconstruct user behaviour, identify communication channels, and uncover hidden and encrypted evidence.

---

## Objectives

- Perform forensic investigation on compromised systems
- Reconstruct user activity across multiple devices
- Identify communication, concealment, and malware usage
- Extract and analyse digital evidence using forensic tools

---

## Evidence Overview

The investigation involved analysing multiple Windows 10 systems, including disk images and memory captures, to identify key artefacts and behavioural patterns.

The case required identifying suspect roles, communication methods, and digital evidence supporting criminal activity.

---

## Tools and Technologies

| Tool | Purpose |
|---|---|
| Autopsy | Disk forensics and artefact recovery |
| Magnet AXIOM | Advanced artefact parsing and memory analysis |
| ChromeCacheView | Browser cache extraction |
| DB Browser | SQLite database analysis |
| TrueCrypt | Encrypted volume analysis |
| Image Steganography Tools | Hidden data extraction |
| Notepad++ / JSON Viewers | Log and artefact validation |

---

## Techniques Applied

| Technique | Description |
|---|---|
| Disk Forensics | NTFS, registry, prefetch, and user activity analysis |
| Memory Analysis | Runtime artefacts and log examination |
| Encryption Analysis | TrueCrypt container investigation |
| Steganography Detection | Hidden data extraction from image files |
| Malware Analysis | Quasar RAT behaviour and log analysis |
| Web and Communication Analysis | Discord and ProtonMail artefact recovery |

---

## Key Findings

- Identified roles of multiple suspects involved in coordinated criminal activity
- Extracted hidden data from steganographic image files
- Decrypted encrypted volumes containing operational data
- Detected and analysed Quasar RAT malware used for remote surveillance
- Correlated browser activity, communication logs, and file artefacts
- Reconstructed a full timeline of events across multiple systems
